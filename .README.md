# 🛍️ Aliya Cloth Store App

A modern e-commerce web application for selling fabrics from Jaipur, India.

## ✨ Features
- 📦 Browse products with filtering (search, print, category)
- 🛒 Shopping cart with localStorage persistence
- 💳 Secure checkout with cash on delivery
- 📱 Fully responsive design
- ⚡ Built with TypeScript & Vite

## 🛠️ Tech Stack
- **Frontend:** TypeScript, Vite, Vanilla JavaScript
- **Backend:** Supabase (PostgreSQL database)
- **Styling:** CSS3
